# BMI Calculator

## Overview
This Python project calculates a user's Body Mass Index (BMI) based on their weight and height and classifies the result into standard BMI categories.

## Technologies Used
- Python
- Jupyter Notebook

## Features
- User input for name, weight, and height
- BMI calculation
- BMI category classification
- Simple conditional logic implementation

## BMI Categories
- Underweight
- Normal Weight
- Overweight
- Obese
- Severely Obese
- Morbidly Obese
